# Enhance_Booth

BOOTHの検索結果や商品表示をカスタマイズし、ノイズを減らしてブラウジングを効率化するブラウザ拡張機能です。

## 主な機能

### 1. 検索フィルタリング
- **NGワード（タイトル）**: 指定したキーワードをタイトルに含む商品を非表示にします。
- **NGショップ**: 指定したショップの商品を非表示にします。
- **URLクエリ自動付与**: BOOTH標準の検索機能（`except_words`）に、登録したNGワードを自動的に追加して検索を実行します。

### 2. 表示カスタマイズ
- **2列グリッドレイアウト**: 検索結果を視認性の高い2列のグリッド表示に変更できます（ON/OFF切替可能）。
- **タイトルのノイズ除去**: 商品名に含まれる「【セール中】」「✨」などの装飾・宣伝的なワードを非表示にし、本来の商品名を見やすくします。
- **特定セクションの非表示**: 商品詳細ページ下部の「『VRChat』の人気作品」セクションを非表示にできます。

### 3. 操作性の向上
- **クイックNG登録**: 商品カード上の「×」ボタンから、即座にそのショップをNG登録できます。
- **同一ショップの整理**: 検索結果に同一ショップの商品が複数並んでいる場合、それらをまとめて表示の末尾に移動させ、検索の多様性を確保します。
- **動的読み込み対応**: ページをスクロールして商品が追加読み込みされた際も、自動的にフィルタリングやレイアウト調整を適用します。

## インストール方法

1. このリポジトリをクローンまたはZIPでダウンロードし、解凍します。
2. Google Chrome等のChromium系ブラウザで `chrome://extensions/` を開きます。
3. 右上の「デベロッパー モード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、解凍したフォルダを選択します。

## 使い方

- **設定変更**: ブラウザのツールバーにある拡張機能アイコンをクリックして、各機能のON/OFFやNGワードの管理を行えます。
- **ショップのブロック**: 
  - 商品カードに表示される「×」ボタンをクリック。
  - または、ショップ名のリンクを右クリックしてコンテキストメニューから追加（※background.jsの実装に依存）。
